The extension contains all examples from the AI First Programming book series, organized hierarchically:

- **Books**: Different programming languages (e.g., Python, Java)

### Getting Started

1. Open the **AI First Books** panel in your sidebar
2. Navigate to the **book** for the language you're learning
3. Expand the **section** that matches what you're reading
4. Browse through **chapters** to find relevant examples

The content is organized to match the structure of the AI First Programming books, making it easy to find examples that correspond to what you're currently studying.
