### Navigation Steps

1. **Expand Sections**: Click on section names to reveal chapters
2. **Expand Chapters**: Click on chapter names to reveal examples
3. **Click Examples**: Click on any example to view its details

### What You'll See

When you click on an example, a new panel opens beside your editor showing:

- **Example Title & Description**: What the example teaches
- **Prompt Text**: The exact question asked to generate the code
- **Code Response**: The generated code with syntax highlighting
- **Copy Buttons**: One-click copying for both prompts and responses

The example opens beside your editor so you can reference it while coding!
